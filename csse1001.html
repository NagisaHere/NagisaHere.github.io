ECP

First core course for any Engineer.

You will program a CLI application in Python. This course will introduce you to procedural programming, which includes the use of loops, conditional statements, and functions that have order.
You will also learn basic data structures (lists, sets, dictionaries), and an introduction to further programming techniques such as recursion and dynamic programming.
GUI based programming may change with varying semesters.

Difficulty rating is definitely lower if you have prior programming experience.
