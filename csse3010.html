Core course for Electrical Engineering, teaching programming of advanced microcontrollers.

Use of the STM32 microcontroller family, interfacing with common peripherals (LEDs, SSD, Radio Module).
Learn how to program in freeRTOS to implement state machines, alongside common error encoding schemes such as hammming encoding.

This is a high workload course, so it is once again recommended to take this with an easy course.
